# 飞猪品牌会员自运营平台 Demo

这是一个用于内部演示的前端 Demo，主题为“希尔顿中国旗舰店在飞猪做品牌会员自运营”。

目标不是做真实上线系统，而是把下面这条主链路跑通：

`选模板 -> 创建活动 -> 挂载触点 -> 前台预览生效 -> 数据分析看结果`

## 技术栈

- React
- TypeScript
- Vite
- React Router
- 本地 mock data + 前端内存状态

## 已实现模块

- 工作台
- 会员身份
- 权益展示
- 运营模板
- 创建活动
- 触点管理
- 数据分析
- 系统设置
- 前台预览

## 联动说明

- 新建活动后，会同步出现在工作台活动列表
- 新建活动后，会自动进入触点管理挂载关系
- 新建活动后，会出现在数据分析活动效果表
- 修改权益展示文案后，前台预览页同步更新
- 切换角色后，创建活动页按钮权限表现不同
- 切换会员身份和触点后，前台预览内容会变化

## 推荐演示路径

1. 进入“工作台”，介绍品牌经营总控台
2. 打开“运营模板”，选择“新会员首单激活”
3. 进入“创建活动”，填写文案并发布
4. 回到“工作台”，确认新活动已出现
5. 打开“触点管理”，确认活动已挂载到目标触点
6. 打开“前台预览”，切换身份和触点，看展示变化
7. 打开“数据分析”，确认新活动已进入效果表
8. 打开“系统设置”，切换角色说明共管模式
