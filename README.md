# 飞猪品牌会员智能经营平台

这是一个用于内部演示的前端 Demo，主题为“希尔顿中国旗舰店在飞猪做品牌会员智能经营”。

本轮升级重点是把 Demo 从“会员配置后台”升级成“AI 智能经营驾驶舱 + 会员营销加速器”，让演示链路变成：

`AI 诊断 -> 选择模板/策略 -> 创建活动 -> 营销加码 -> 渠道配置 -> 触点生效 -> 前台预览 -> 数据分析`

## 技术栈

- React
- TypeScript
- Vite
- React Router
- 本地 mock data + 前端内存状态

## 已实现模块

- 工作台
- 会员身份
- 权益展示
- 运营模板
- 营销活动
- 渠道管理
- 触点管理
- 数据分析
- 系统设置
- 前台预览

## 本轮升级亮点

- 首页升级为 `AI 智能经营驾驶舱`
- 新增 `当前运营阶段识别`、`AI经营诊断`、`一键采用策略`
- 新增 `营销活动` 模块，支持预算、资源位、效果预估
- 新增 `渠道管理` 模块，区分免费渠道与付费渠道
- 数据分析升级为 `会员经营分析 + 渠道投放分析`
- 前台预览新增 `基础模板模式 / 营销加码模式` 切换

## 联动说明

- 首页 AI 推荐策略可直接跳转到模板活动或营销活动创建页
- 新建模板活动后，会同步出现在工作台、触点管理和数据分析中
- 新建营销活动后，会同步出现在工作台、营销活动列表、渠道管理、前台预览和数据分析中
- 修改权益展示文案后，前台预览页同步更新
- 在渠道管理中调整免费/付费渠道后，预算与活动渠道组合同步更新
- 切换角色后，模板活动页和营销活动页按钮权限表现不同

## 推荐演示路径

1. 进入“工作台”，先讲当前经营阶段、AI 诊断和一键推荐策略
2. 点击“首住激活加速方案”，进入模板活动创建页
3. 返回工作台，再进入“营销活动”，展示预算、资源位和加码效果
4. 打开“渠道管理”，展示免费渠道与付费渠道组合
5. 打开“触点管理”，确认活动已经挂到对应阵地
6. 打开“前台预览”，切换基础模板模式和营销加码模式
7. 打开“数据分析”，展示活动效果、增量GMV 和渠道 ROI
8. 最后在“系统设置”说明飞猪小二和商家的共管模式
